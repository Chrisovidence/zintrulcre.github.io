Google Kick Start 2019 Round A

March 26, 2019
Kick Start

Training (7pts, 13pts)

解析

一道简单的数组题，简单翻译一下就是一共有N个人，从中选P个人，假设这P个人的 skill rating 分别是 rating[i], rating[i + 1], ... , rating[j]，计算这P个人中 skill rating 的最大值 max(rating) 与其他人的 skill rating 的差值 max(rating) - rating[i] 之和，

\( \sum_{i}^{j}\)max(rating) - rating[i]。

#1

先对数组排序，然后在长度为N的有序数组中遍历长为P的所有连续子数组，计算子数组中的最大值与其他值的差值之和，\( \sum_{i}^{j - 1} \) rating[j] - rating[i]。

这样做时间复杂度是O(NlogN + NP)，空间复杂度是O(1)。

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    int c = 1;
    int T;
    std::cin >> T;
    for (int t = 0; t < T; ++t) {
        int N, P;
        int min_hour = 1000000000;
        std::cin >> N >> P;
        std::vector<int> rating(N, 0);
        for (int n = 0; n < N; ++n)
            std::cin >> rating[n];
        std::sort(rating.begin(), rating.end());
        for (int i = P - 1; i < N; ++i) {
            int sum = 0;
            for (int j = i - P + 1; j < i; ++j)
                sum += rating[i] - rating[j];
            min_hour = std::min(min_hour, sum);
        }
        std::cout << "Case #" + std::to_string(c) + ": " + std::to_string(min_hour) << std::endl;
        ++c;
    }
    return 0;
}

#2

#1的做法只能通过visible test set，因为遍历长为P的子数组浪费了太多时间，可以将上面的公式简化为

\( \sum_{i}^{j - 1} \) rating[j] - rating[i] = rating[j] * (j - i - i) - \( \sum_{i}^{j - 1} \) rating[i]

为了避免重复计算 \( \sum_{i}^{j - 1} \) rating[i]，可以用一个长为N+1的数组将原始数组的前缀和保存下来，这样每次直接计算 prefix[j] - prefix[i] 就能得到 \( \sum_{i}^{j - 1} \) rating[i] 了。

这样做时间复杂度是O(NlogN + N)，空间复杂度是O(N)。

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    int c = 1;
    int T;
    std::cin >> T;
    for (int t = 0; t < T; ++t) {
        int N, P;
        int min_hour = 1000000000;
        std::cin >> N >> P;
        std::vector<int> rating(N, 0);
        for (int n = 0; n < N; ++n)
            std::cin >> rating[n];
        std::sort(rating.begin(), rating.end());
        
        std::vector<int> prefix_sum(N + 1, 0);
        for (int i = 0; i < N; ++i)
            prefix_sum[i + 1] = prefix_sum[i] + rating[i];

        for (int i = P - 1; i < N; ++i) {
            int sum = rating[i] * (P - 1) - (prefix_sum[i] - prefix_sum[i - P + 1]);
            min_hour = std::max(0, std::min(min_hour, sum));
        }

        std::cout << "Case #" + std::to_string(c) + ": " + std::to_string(min_hour) << std::endl;
        ++c;
    }
    return 0;
}
