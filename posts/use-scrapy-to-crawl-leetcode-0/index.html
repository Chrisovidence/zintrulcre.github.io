Use Scrapy and Splash to Crawl LeetCode 0

使用Scrapy和Splash爬取LeetCode（0）：环境配置

使用到的环境：

  • macOS 10.14.2
  • PyCharm 2018.3.2
  • Visual Stuodio Code 1.30.1
  • Chrome 71.0.3578.98
  • Python 3.7.2
  • Scrapy 1.5.1
  • Docker 2.0.0
  • Splash

环境的配置

  1. 下载安装最新版的 Python3
  2. 使用pip指令安装最新版的Scrapy pip3 install scrapy
  3. 下载安装最新版的 Docker
  4. 使用Docker下载安装最新版的Splash docker pull scrapinghub/splash
  5. 安装能够方便地查看网页xpath的chrome的插件xpath helper和禁止网页加载JS的插件Toggle JavaScript
